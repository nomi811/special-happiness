Important HTML tags

This page is designed as a quick reference for some of the common tags for HTML.

The tables below don't indlude all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make bold text <em>technical sophistication</em> technical sophistication
a link to make a link <a href="https://learnenough.com/">Learn Enough</a> Learn Enough
img image to make an image <img src="https://bit.ly/1MZAfuQ" ALT="Michael Hartl"> Michael Hartl
